Dear GitHub,

Thank you for repositioning yourself in my life. You've become an essential part of my life. You can easily read me and understand my needs. Despite the push and pull of our relationship, I'm a better person because of you. I would even go to a remote island with you, with only a fork in hand. That would definitely merge us for life. It's almost as if you'e my clone. I request that this relationship lasts forever.

Love,

Kaylin

  • repo : file history of a project
  • readme.md : file that can provide information or instructions of a project
  • git push origin master : push changes from local working directory to github
  • pull : fetches and merges changes on the remote server to local working directory
  • git remote add origin url : set up connection to remote repository
  • fork : allows you to make a copy of a repository and make changes without affecting orignial branch
  • merge : merges two branches
  • git clone url : clone a repository to local working directory
  • pull request : request your changes to be accepted by project owner
