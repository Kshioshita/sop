Dear Git,

Initially, I was hesitant about our relationship. I didn't want to commit too much to you, but you kept leaving me messages. I soon realized that the status of our relationship was evolving. I kept wanting to add you to my future. The log of the time we spent together kept growing. The list of differences between us kept getting smaller. I was no longer able to branch off in other directions. I only wanted to create a new branch with you. I never found myself checking out other people. I feel like going back in time when I'm with you. We should merge our lives together. You've helped me delete painful branches from my past. I will never push you away again.

Love,

Kaylin

  • init : initialize a git project
  • git commit: permanent save, like printing
  • -m : add message to commit
  • git status : what stage the files are at
  • git add : add files to staging
  • git log : log of changes including author and date
  • git diff : chacks differences between working directory files and repository
  • git branch : list of branches
  • git branch branchname : creates new branch
  • git checkout branchname : switch branches
  • git checkout hash# : get previous version
  • git branch merge : merge branches
  • git branch -D : delete branch
  • git push -u origin master : save changes to remote repository
